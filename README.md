# 📊 Interactive Sales Dashboard

An interactive data visualization tool built with **Python** and **Streamlit**. This dashboard provides a comprehensive overview of sales performance, allowing users to drill down into specific dates, product categories, and geographic regions.

## ✨ Features

* **Real-time KPIs:** Instantly view Total Sales, Average Sales, and Transaction counts.
* **Sidebar Controls:** Filter the entire dashboard by:
    * **Date Range:** Specific start and end periods.
    * **Category:** Electronics, Clothing, and Food.
    * **Region:** Nairobi, Mombasa, and Kisumu.
* **Interactive Visuals:** * **Sales Over Time:** A multi-colored line chart for trend analysis.
    * **Category Breakdown:** Bar charts to compare department performance.
    * **Sales Distribution:** Histograms to see the frequency of sales values.
    * **Regional Statistics:** Boxplots to analyze sales variance by location.

## 🛠️ Built With

- [Streamlit](https://streamlit.io/) - Web framework for Data Science.
- [Pandas](https://pandas.pydata.org/) - Data manipulation and analysis.
- [Plotly Express](https://plotly.com/python/plotly-express/) - Interactive plotting library.
- [NumPy](https://numpy.org/) - Numerical computing.

## 🚀 Getting Started

### Prerequisites

Ensure you have Python installed, then install the necessary libraries:

```bash
pip install streamlit pandas numpy plotly
```
How to run the app
```bash
streamlit run app.py
```
